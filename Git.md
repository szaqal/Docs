# Git

## Clone Tag

```
git clone --branch <tag> <repo>
```

## Delete remote GIT tag

```
git push --delete origin v-xxxx-release
```

## Change commit message (last)

```
git commit --amend
```

## Compare Branches

```
git diff feature-x..develop
```

## File changes history

```
git log <file>
```

## Logs

```
git log --stat
git log -p
```
